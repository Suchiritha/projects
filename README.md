# Student Management System (Python + SQL)

This project is a simple Student Management System developed using Python and SQLite database.

## Project Description
The application helps to manage student academic records efficiently.  
It allows users to store student details such as roll number, name, and marks in a database.

## Features
- Add Student Record
- View All Student Records
- Delete Student Record

## Technologies Used
- Python
- SQLite (SQL Database)

## How to Run the Project
1. Download the file `student_management.py`
2. Open terminal or command prompt
3. Run the program using:

```bash
python student_management.py
