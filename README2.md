# Simple Encryption & Decryption Tool (Python)

This project is a simple encryption and decryption tool developed using Python.

## Project Description
The tool converts normal text into an encoded format and then decodes it back to the original message.  
It demonstrates the basic concept of data confidentiality and secure data transformation.

## Features
- Encrypt text using Base64 encoding
- Decrypt encoded text back to original format

## Technologies Used
- Python

## How to Run
1. Download the file `simple_encrypt_decrypt.py`
2. Run the program using:

```bash
python simple_encrypt_decrypt.py
